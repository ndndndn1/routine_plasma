# EFIT-mini: An Embedded, Multi-task Neural Network-driven Equilibrium Inversion Algorithm

- **Authors:** Guohui Zheng, Songfen Liu, Huasheng Xie, Hanyue Zhao, Yapeng Zhang, Xiang Gu, Zhengyuan Chen, Tiantian Sun, Yanan Xu, Jia Li, Dong Guo, Renyi Tao, Youjun Hu, Zongyu Yang
- **Venue / Date:** arXiv preprint, submitted 25 Mar 2025
- **Link:** https://arxiv.org/abs/2503.19467
- **Routine date:** 2026-05-13 (within 2-year window: 2024-05-13 .. 2026-05-13)

## Keyword matches (>=2 required)
1. **Magnetic Equilibrium Reconstruction** — the core task: infer poloidal flux ψ(R,Z), toroidal current density Jt, and plasma boundary from magnetic diagnostics and PF coil currents.
2. **Equilibrium Fitting** — EFIT-mini retains a least-squares Grad–Shafranov fit step and uses a neural network as an accelerator inside the EFIT-style inversion loop.
3. **Inverse Problem Solving** — equilibrium reconstruction is a classic ill-posed inverse problem (sparse magnetic measurements → 2D flux map).
4. **Magnetic Flux Function** — the network outputs polynomial coefficients of p′(ψ) and ff′(ψ), the source functions of the Grad–Shafranov equation.

## Methods

**Problem.** Solve the Grad–Shafranov equation
Δ*ψ = −μ₀ R Jt(R, ψ) = −μ₀ R² p′(ψ) − ff′(ψ),
where Δ* = R ∂/∂R (R⁻¹ ∂/∂R) + ∂²/∂Z², subject to consistency with magnetic probe / flux loop measurements and PF coil currents. Traditional EFIT iterates between an elliptic solve and a least-squares fit of p′ and ff′ until convergence; this is slow and can fail to converge in real time.

**Architecture (EFIT-mini).** EFIT-mini replaces the bottleneck steps of EFIT with a single multi-task neural network:
- *Inputs:* PF/CS coil currents and magnetic probe + flux loop measurements (e.g., 68 channels on HL-3 in the predecessor EFITNN work).
- *Outputs (multi-task heads):*
  1. Eight scalar plasma parameters (Ip, βp, li, etc.).
  2. The 2D poloidal flux ψ(R,Z) on a 129×129 grid, from which the LCFS and X-point are extracted.
  3. The 2D toroidal current density Jt(R,Z) on the same grid.
  4. Polynomial coefficients for p′(ψ) and ff′(ψ).
- *Multi-task weight sharing* improves accuracy by up to ~32 % relative to single-task baselines because the shared encoder is regularized by physically related targets.

**Hybrid loop.** The neural-network outputs are not used end-to-end. EFIT-mini feeds the predicted p′, ff′ and boundary into a conventional least-squares constraint on the magnetic measurements (a "physical prior"); the GS equation is then re-solved on the grid. This keeps the answer consistent with raw diagnostics rather than only with the training distribution.

**Training data.** Equilibria from EXL-50U (validation) and an HL-3 cohort (the predecessor EFITNN work used 1,159 discharges). Targets are generated by traditional EFIT/EFITNN runs.

**Speed.** Single-slice 129×129 inference in 0.11 ms (NN only) and 0.36 ms (full algorithm including the LS step), three orders of magnitude faster than CPU EFIT.

## Conclusions

- A multi-task NN can replace the most expensive parts of EFIT (boundary identification, p′/ff′ profile guessing) while a thin physics-based wrapper enforces consistency with diagnostics.
- Validated on EXL-50U: >98 % LCFS-area overlap with conventional equilibrium reconstruction.
- Sub-millisecond latency makes the scheme suitable for real-time tokamak control loops on embedded hardware ("embedded" in the title refers to deployment on plasma-control-system FPGAs/GPUs).
- The combination of *learned priors* and *least-squares physical fit* mitigates the usual generalization risk of purely data-driven equilibrium solvers.

## Code

No public code repository is referenced in the paper. `reference_code.py` in this folder is an original, self-contained implementation of the core idea — a multi-task MLP that maps simulated magnetic signals to (Ip, βp, li, ψ-map, Jt-map) and uses a thin Grad–Shafranov residual head as a physics regularizer.
